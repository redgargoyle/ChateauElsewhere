# SpriteLibrary Browse Layer

This folder is the central place to browse current character sprite work.

## What Is Gathered Here
- `Characters/<GuestID>_<Name>/PanicFrames/ReferenceLockedOrGuestReference/` contains copied review frames for the current right-style panic set.
- `ReviewSheets/` contains generated contact sheets so you can visually scan each guest's generated candidates and project reference sprites without hunting through scattered folders.
- `Indexes/ALL_PNG_LOCATIONS.csv` lists PNGs found in the known art locations.
- `ROSTER_AND_ALIASES.md` resolves guest-number/name confusion.

## What Is Not Moved
Runtime/source art remains in its original locations, including `Assets/Art/Characters`, `Assets/AnimationLibrary`, `Assets/GeneratedSpriteLibrary`, and `Assets/GeneratedSpriteLibraryStyleMatched`. This browse layer is non-destructive.

## Current Panic Rule
For the latest right-style panic frames, use the central panic-frame folders here or the original source folders listed in each character README. Avoid raw chroma sheets and old `panic_hands_up` folders unless explicitly reviewing old work.
